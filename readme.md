# 📚 Book Search App
### Приложение для поиска книг с использованием открытой библиотеки API Open Library.


## 🚀 Технологии
- Ванильный JavaScript (ES6+) - чистый JS без фреймворков
- Компонентный подход - собственная реализация компонентной архитектуры
- Open Library API - источник данных о книгах
- CSS3 - современная стилизация с анимациями
- on-change - реактивное управление состоянием
- Rollup - сборка проекта
- Live Server - локальный сервер для разработки-
- ESLint - линтинг кода

## 📋 Предварительные требования
- Node.js (версия 14 или выше)
- npm (менеджер пакетов Node.js)

## 🔧 Установка и запуск
- Клонируйте репозиторий
```git clone https://github.com/StanleyIG/java-striptease-advanced.git```  
```cd java-striptease-advanced```
- Установите зависимости
```npm install```
- Соберите проект
```npm run build```


### В одном терминале запустите сборку в режиме watch
```npm run build```

### В другом терминале запустите сервер
```npm start```
📦 Скрипты npm
```npm run build``` - сборка проекта с Rollup в режиме наблюдения

```npm start``` - запуск локального сервера с автоперезагрузкой либо с помощью расширения
